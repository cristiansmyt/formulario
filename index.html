Questionário

1) Responda as perguntas abaixo:

a. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

O campo "Placeholder" é utilizado para indicar ao navegador o texto que deverá aparecer escrito no campo do formulário, na maioria das vezes utilizado para indicar ao usuário como deve ser preenchido aquele campo em específico ou a função daquele campo de formulário.


b. Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"

Esse é um recurso realmente útil de rótulos de formulários HTML que facilitam o clique na opção desejada, especialmente em dispositivos de tela pequena, como smartphones. Além da acessibilidade, esse é outro bom motivo para configurar corretamente os <label> elementos em seus formulários.


c. Na tag <input> para que serve o atributo "value"?

Nos inputs text o value é o que a pessoa digita, nos checks e radios são nomes que nós colocamos para fazer sentido quando recebermos essa informação na outra ponta.


d. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Para usar este tipo de campo de um formulário, temos que fornecer o valor "radio" ao atributo type da tag <input>. Também devemos fornecer o atributo nome, de tal modo que todas as opções, todo o grupo de botões rádio, devem ter o mesmo o nome.


e. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

Um elemento **HTML <label> **representa uma legenda para um item em uma interface de usuário. "for" é o atributo de entrada para qual o <label> se destina. Seu valor deve ser igual ao valor do atributo id do elemento controle de entrada


f. Na tag <option> para que serve o atributo "selected"?

O elemento select permite uma ou mais opções em um menu onde cada opção é representada por um elemento option.


g. Nas tags de formulário para que serve o atributo "required"?

Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.


h. Na tag <form> para que serve o atributo "action"?

O action atributo especifica para onde enviar os dados do formulário quando um formulário é enviado.


i. Na tag <form> para que serve o atributo "method"?

O method atributo especifica como enviar dados de formulário (os dados de formulário são enviados para a página especificada no action atributo).


j. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

O valor digitado e o "post"


k. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

O valor deve ser o "id" de um elemento <datalist> no mesmo documento. O navegador mostrará apenas opções que são valores válidos para este elemento.
